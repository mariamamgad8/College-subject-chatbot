# College Subjects Chatbot

## Project Idea
A machine learning chatbot for college subjects that classifies student questions into intents and returns the proper educational response.

## Steps to Run
1. Install requirements:
   pip install -r requirements.txt

2. Train the model:
   python -m src.train

3. Run the chatbot:
   python app.py

## Output Files
- Trained model saved in `models/`
- Label encoder saved in `models/`
- Plots saved in `reports/generated_plots/`
